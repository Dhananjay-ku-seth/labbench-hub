# LabBench

**Your engineering lab, in the browser.**

The brand hub for a suite of interactive engineering tools — real-time DSP, PID control, digital
logic, and communication systems, each runnable with zero installs.

**Live:** https://labbench-hub.vercel.app/

## The suite

| Tool | Demo | Source |
|---|---|---|
| DSP Signal Lab | https://dsp-signal-lab.vercel.app/ | https://github.com/Dhananjay-ku-seth/dsp-signal-lab |
| PID Control Playground | https://pid-control-playground.vercel.app/ | https://github.com/Dhananjay-ku-seth/pid-control-playground |
| Logic Circuit Simulator | https://logic-circuit-sim.vercel.app/ | https://github.com/Dhananjay-ku-seth/logic-circuit-sim |
| Comms Simulator | https://comms-simulator-pi.vercel.app/ | https://github.com/Dhananjay-ku-seth/comms-simulator |

## Model

Free: all 4 tools, full functionality, no account required.
Pro (coming soon): saved sessions, exports, and a classroom mode for instructors.

## Tech

React + TypeScript + Vite, deployed on Vercel. No UI libraries.

## Run locally
```sh
npm install
npm run dev
```
